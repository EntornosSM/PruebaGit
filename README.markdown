PruebaGit
=========

PruebaGit es un proyecto resultado del tutorial de introducción a Git,EGit y GitHub publicado en Internet.
El objeto es realizar un flujo de trabajo muy básico que permita iniciar nuevos proyectos con Git, gestionarlos a un nivel básico y subirlos a GitHub

Instalacion
-----------

Para la correcta instalación y funcionamiento del proyecto hará falta al menos:

*Git

*Eclipse

*Plugin EGit para Eclipse

*Una cuenta en GitHub
